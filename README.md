# Formulário com Node.js e Banco de Dados

Este projeto consiste em um **formulário web** que envia dados para um **servidor Node.js**, onde essas informações são **processadas e armazenadas em um banco de dados**.

O objetivo do projeto é praticar a integração entre **frontend e backend**, além da persistência de dados.

---

## Tecnologias Utilizadas

- **HTML5** – Interface do formulário  
- **Node.js** – Servidor backend  
- **Express.js** – Framework para rotas e requisições HTTP  
- **Banco de Dados** – Armazenamento das informações enviadas  
- **JavaScript** – Lógica da aplicação
